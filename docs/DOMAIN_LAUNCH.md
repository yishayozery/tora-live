## שלב 2 — קנייה (15 דק׳)

### 🥇 רשם מומלץ: **ISOC ישראל** (האגודה הרשמית)

**למה ISOC?**
- ✅ הזול ביותר (~₪50/שנה לעומת ~₪80 ברשמים אחרים)
- ✅ הרשמי לישראל (אין מתווך)
- ✅ תמיכה בעברית
- ✅ אין upsells מעצבנים
- ❌ UI לא מודרני, אבל פשוט

**קישור**: https://www.isoc.org.il/domains

**צעדים**:
1. הכנס לאתר ISOC
2. "רישום דומיין" → הקלד `tora-live`
3. בחר `.co.il` (או `.org.il` אם אתה מתכנן לרשום עמותה)
4. הרשמה / כניסה (תצטרך תעודת זהות + פרטי קשר)
5. תשלום — כרטיס אשראי, ~₪50 לשנה
6. אישור — תקבל מייל תוך 5 דקות

### 🥈 חלופה זרה: **Cloudflare Registrar**

**למה Cloudflare?**
- ✅ מחיר עלות (לא מרוויח, מעביר אותך לרישום במחיר רשמי) — ~₪40/שנה
- ✅ DNS המהיר בעולם (מובנה)
- ✅ הגנת DDoS חינמית
- ✅ UI מודרני באנגלית
- ❌ דורש כרטיס אשראי בינלאומי
- ❌ לא רוכש `.co.il`! (רק `.com`/`.org`/`.net`/`.io` וכו')

**מתי לבחור**: רק אם בחרת `.com` או `.org`.

### 🥉 לא ממליץ: **GoDaddy / Hostinger**

- יקרים יותר (₪80-100)
- מנסים למכור לך 50 שירותים נוספים
- UI מבלבל
- שירות "Domain Privacy" שמחייב ₪40 נוספים — שלא תצטרך

## שלב 3 — חיבור הדומיין ל-Vercel (15 דק׳)

לאחר שקנית את הדומיין:

### א. הוסף את הדומיין ב-Vercel

1. כנס ל-https://vercel.com/yishayozerys-projects/torah-live/settings/domains
2. הזן `tora-live.co.il` → "Add"
3. Vercel יציג לך **2 רשומות DNS** שצריך להוסיף:
   ```
   Type: A      Name: @     Value: 76.76.21.21
   Type: CNAME  Name: www   Value: cname.vercel-dns.com
   ```
   (הערכים יכולים להיות שונים אצלך — תעתיק את מה ש-Vercel מציג בדיוק)

### ב. הוסף את הרשומות ב-ISOC (או הרשם שלך)

1. כנס לחשבון ב-ISOC → "ניהול הדומיין שלי" → `tora-live.co.il`
2. "ניהול DNS" → "הוסף רשומה"
3. הוסף את ה-A record:
   - Type: A
   - Host: `@` (או השאר ריק)
   - Points to: `76.76.21.21`
   - TTL: 3600
4. הוסף את ה-CNAME:
   - Type: CNAME
   - Host: `www`
   - Points to: `cname.vercel-dns.com`
   - TTL: 3600
5. שמור

### ג. בדיקה (חכה 5-30 דקות)

```bash
nslookup tora-live.co.il
# אמור להחזיר 76.76.21.21
```

או פתח https://tora-live.co.il בדפדפן. אם זה פותח את האתר → ✓

### ד. SSL אוטומטי

Vercel מנפיק תעודת SSL של Let's Encrypt באופן אוטומטי. תוך כמה דקות תראה ✓ ירוק ב-Vercel dashboard ליד הדומיין.

### תיקון בעיות נפוצות:

**הדומיין לא נטען**:

- חכה 10-15 דק׳ נוספות
- וודא ש-DNS מצביע נכון

**התחברות לא עובדת**:

- Redeploy

**מיילים לא נשלחים**:

## ❓ שאלות נפוצות

**Q: מה ההבדל בין `.co.il` ל-`.org.il`?**
- `.co.il` = מסחרי (יותר נייטרלי, מתאים לסטארט-אפ)
- `.org.il` = ארגון ללא רווח (אם תרשום עמותה — תעבור ל-`.org.il`)
- היום בחר `.co.il`. אם תהפוך לעמותה — תוכל להוסיף `.org.il` במקביל.

**Q: צריך לקנות גם `.com`?**
- לא חובה היום. אם הפרויקט יגדל — שווה לקנות הגנתית.
- ~₪80/שנה.

**Q: מה לגבי trademark?**
- `TORA_LIVE` כשם — חינם, אין רישום משפטי דרוש.
- אם תרצה הגנה — רישום סימן מסחרי בישראל ~₪1,500.
- לא דחוף.

**Q: אם אני קונה ב-ISOC, מי הבעלים?**
- אתה. ת.ז שלך רשומה.
- חידוש שנתי באותה דרך.
- אפשר להעביר בעלות בעתיד (תהליך פשוט).
